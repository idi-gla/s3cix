# Setup

Each course has instructions for the pre-requisites required for installation in `readme.md`.


### Using `uv`

**We recommend you use the `uv` approach to install requirements where possible.** This will install the packages required in a separate environment that won't interfere with your existing Python install; it's also very fast.
* Run `uv run jupyter <notebook>.ipynb` to install everything and start up the notebook to check it works.
